Welcome to the Training Wizard Tour!

This page introduces you to the 3 components of the Training Wizard.    To get started with the tour, simply click on the top panel header below.

This component gives you access to the following features and functions:
This is the icon that will be used to identify your training. Clicking the icon brings up the customization dialog.
This loads the page you are currently viewing.
This loads the Training Configuration page.
Clicking on this icon brings up the file upload dialog. From there you can identfy and upload the documents used in this training.
Clicking on this icon puts the training in preview mode, removing all of the edit and configuration elements.
Clicking on this icon deletes the training.
Clicking on this icon switches to fullscreen mode.
Clicking on this icon closes the training.
Clicking on this icon loads the comment page. This is only available in Preview mode or when the training status is 'In Review' or 'In Production'.
This component provides page navigation.    When documents are uploaded to a training, the document name is added to this component.
This is the content viewing component.    Selecting entries in the Table of Contents loads the selected page here.
